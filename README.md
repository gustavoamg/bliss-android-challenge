# Bliss Android Challenge

This project is a showcase development for a technical challenge requested by the recruitment process
from Bliss Applications. The exercise description can be found in the document [Bliss Recruitment - Challenge for Android.pdf](<./Bliss Recruitment - Challenge for Android.pdf>) on project root folder.

The following stack is used in this development:


| Layer | Technology  |  Version   |
|---|:-----------:|:----------:|
| Language | Kotlin |   2.4.0    |
| UI Framework | Jetpack Compose | 2026.06.01 |
| Image Downloading | Coil |   3.5.0    |
| CDI | Hilt |   2.60.1   |
| API Consumption | Retrofit |   3.0.0    |
| Local Persistence | Room |    2.8.4     |
| Navigation | Navigation 3 | 1.1.4 |
